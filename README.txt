Este es el proyecto final para la clase de lenguaje de Marcas
de la promoción 2022-2024 del CEINMARK (Santander).

Estos son los requisitos del proyecto:

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Elaboración de un sitio web utilizando HTML y CSS solamente como lenguajes con los que codificar.

El sitio web tendrá elementos originales íntegramente (no podrán incluirse materiales de otros autores).

El sitio web tendrá las siguientes páginas como mínimo:

1) Home: página principal donde se encontrará el acceso a las diferentes áreas de la página así como una pequeña presentación de la misma.

2) Sobre mí: página en la que se expondrá vuestro perfil profesional incluyendo:
  2.1) Puesto de trabajo al que aspiráis.
  2.2) Resumen (Lo que se os da bien y lo que creéis que tenéis o podéis desarrollar que la empresa necesita).
  2.3) Información de contacto y/o redes sociales de uso profesional o académico (esto obviamente puede ser totalmente falso).
  2.4) Estudios realizados (con o sin certificados).
  2.5) Experiencia profesional (pagada o no).
  2.6) Habilidades técnicas (SO, IDEs, Lenguajes conocidos, etc.).
  2.7) Habilidades blandas.

3) Trabajo soñado: página en la que se describe vuestro puesto de trabajo deseado incluyendo:

  3.1) Empresa.
  3.2) Departamento.
  3.3) Características de vuestro supervisor.
  3.4) Beneficios.
  3.5) Salario neto mensual.
  3.6) Plan de carrera.
  3.7) Tareas a realizar.
  3.8) Plan de formación.
  3.9) Modalidad (telemática/presencial/híbrida).

4) Juegos y Hobbies: página en la que hablareis de 3 actividades que os guste realizar en vuestro tiempo libre incluyendo:

  4.1) Descripción de la actividad.
  4.2) Cuanto tiempo hace que tenéis ese hobby.
  4.3) Razón por la que invertís vuestro tiempo en ello.
  4.4) Razón por la que todo el mundo debería invertir su tiempo en ello.

5) IA: página donde mostraréis las herramientas de inteligencia artificial que habéis utilizado para realizar dicha página web incluyendo:

  5.1) Nombre de la plataforma.
  5.2) Tipo de acceso (usuario registrado, invitado, de pago, gratuita, etc.).
  5.3) Elementos generados en la plataforma.
  5.3.1) Nombre del elemento y ubicación en la página.
  5.3.2) Prompt que utilizasteis para generar el elemento.
  5.3.3) Adaptaciones posteriores que necesitasteis realizar.

6) Referencias y agradecimientos: página para añadir las citas de las referencias consultadas que no se corresponden con IAs para el desarrollo de vuestra página web y los agradecimientos que sean pertinentes.

Las citas se generarán con el formato APA. Para ello podéis usar la siguiente herramienta: https://www.grafiati.com/es/
  
Productos a entregar:

1) Web: En el caso de optar por la opción a) un archivo de texto llamado "Acceso" con la url y todas las contraseñas para acceder como administrador a la página. En el caso de la opción b) un archivo comprimido que contenga la carpeta base de la página web

2) Documentación: Cuaderno de bitácora del proyecto que incluya los días que habéis trabajado, las tareas realizadas, así como las sesiones realizadas y su duración. Formato del archivo: PDF

3) Presentación: Vídeo en el que realizaréis una exposición de vuestra página web mostrando tanto el resultado en el navegador como vuestro código. La duración máxima será de 15 minutos con un mínimo de 8. (Os recomiendo subirlo a Google Drive para evitaros problemas de peso del archivo).

Evaluación: 
Será requisito imprescindible la entrega de los 3 productos.
Peso de cada parte: Web (40%) - Documentación (20%) - Presentación (40%).
